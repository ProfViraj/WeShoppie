# WeShoppie
This is WeShoppie repository...
The main motive of this project is to connect the customers
to their trusted local shopkeepers or sellers. This application provides a 
digital platform to both buyer and seller or any other local service provider. 
Customers get an auto-generated to-buy list according to their individual 
order history. Both the end users can anytime see the status of their orders. 
One can compare the cost of the same product available with various 
connected sellers and go for an affordable deal without any quality
compromise. Easy connecting setup and Order Management is a favorable 
aspect of our application.
